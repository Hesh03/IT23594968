SwiftTranslator UI Automation Tests Playwright
This repository contains UIfocused automated tests written using Playwright to validate the performance behavior and stability of the SwiftTranslator web application
The tests are designed as UI characteristic tests not functional backend validation focusing on

Realtime UI updates
UI stability for different input types
Visibility and responsiveness of UI components
